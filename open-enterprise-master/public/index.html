기초

ICT응용입문

◼ 프로그래밍을 처음 시작하는 사람들이 보다 쉽게 학습 할 수 있는, 직관적인 프로그래밍 방식(블럭코딩), 멀티미디어 학습지원, 프로그래밍 공유를 통한 학습용이성을 특성을 가진 앱인벤터를 배운다

◼ 대학 내 다양한 전공을 공부하는 학생들에게 사고력, 문제해결능력, 창의력을 기를 수 있는 기회를 제공하고, 본인의 전공에서 배운 결과를 컴퓨터나 스마트폰을 통해 여러사람들에게 알릴 수 있는 도구로서 프로그래밍을 활용하여 융합형 인재에게 필요한 능력을 제고하도록 돕는다

소프트웨어 입문

◼ MIT 미디어 연구소에서 개발한 언어인 프로세싱으로 컴퓨터 프로그래밍의 코딩을 익힌다.

◼ 프로세싱은 문장 구성이 단순하고, 시각적 표현이 비교적 용이하므로 다른 컴퓨터 언어보다 쉽게 익힐 수 있다.

◼ 특히 3D 그래픽스나 영상처리가 원활하고 쉬우므로 그래픽 처리가 많이 필요한 다양한 전공에서 바로 활용이 가능하다.

◼ 현재 안드로이드 지원이 되기 때문에 결과를 자신의 스마트폰 앱으로 등록하여 사용할 수도 있다.

중급

앱 프로그래밍

◼ 인터넷으로 정보가 소통되는 시대에 인터넷 기본구조에 대한 지식과 웹페이지를 구축하기 위한 기초 기술에 대한 학습을 바탕으로 동적인 웹페이지를 구축할 수 있는 것을 목표로 한다.

◼ 웹페이지 개발언어인 HTML, CSS, Javascript 언어에 대하여 학습한다.

◼ 다양한 프로그래밍 언어를 습득하고 코드 작성을 통하여 논리적인 사고력을 향상 시킨다.

C 프로그래밍

◼ C언어를 사용하여 프로그래밍의 이론적 개념을 배우고 또한 실습을통해.프로그래밍의 전반적인 구조를 이해하도록 한다.

◼ 처음 프로그래밍 언어를 접하는 초보 프로그램 작성자를 주 수강생으로 간주하여 실습 위주의 과제 수행을 통하여 컴퓨터를 이용한 문제 해결 과정을 경험하도록 하며, 더불어 알고리즘 분석 능력을 익힌다

파이썬 프로그래밍

◼ 사람이 생각하는 방식을 그대로 쉽게 표현할 수 있도록 해주고, 문법이 쉬워 빠르게 학습할 수 있으며, 구조가 간결하고 개발속도가 빠른 파이썬 언어를 학습한다.

◼ 파이썬 언어로 응용프로그래밍, 시스템 유틸리티, GUI프로그램, C/C++과의 결합, 웹프로그래밍, 수치연산프로그래밍, 데이터베이스프로그래밍 등을 하는데 적합한 언어이다.

◼ 강의에서는 입출력문, 변수, 자료형, 그림으로 표현하기, 리스트, 조건문, 반목문, 객체지향 개념이해와 활용, 윈도우환경에서 GUI프로그램하기, 파일입출력 등을 다뤄서 파이썬 언어 기초와 중급과정을 학습하여 응용프로그램을 개발해 본다.

◼ 대학 내 다양한 전공을 공부하는 학생들에게 프로그래밍 교육을 통해 융합형 인재에게 필요한 능력을 제고하도록 돕는다.

R이용한 빅데이터 분석

◼ 엄청난 양과 속도로 그리고 다양한 형태의 데이터들이 생성되고 수집되고 있는 가운데 이들 데이터로부터 필요하고 타당한 정보를 수집하고 분석할 필요성이 있다. 본 강의는 이들 데이터를 R programming 언어를 활용하여 필요한 데이터를 수집하고, 다양한 통계 분석을 통해 데이터를 분석하고자 한다.

◼ 패턴인식 기술들을 이용하여 정형화된 데이터들의 관계, 패턴, 추세 등을 분석하는 데이터 마이닝 분석 뿐만 아니라 소셜미디어(트위터, 페이스 북 등)에서 생산되는 비정형 데이터로부터 데이터의 패턴이나 관계를 분석하는 텍스트마이닝(text mining), 감성분석(sentiment analysis), 소셜네트워크분석(social network analysis), 텍스트 클러스터링(text clustering) 방법으로 데이터를 분석하고자 한다.

활용

하이브리드 웹 설계

◼ 이 과목은 인터넷에 관한 역사와 인터넷 기본 활용 및 웹사이트 구축, 웹 페이지 작성 능력을 배양하기 위한 과목이다.

◼ 이 과목은 HTML, CSS, wordpress, Javascript, Jquery, Ajax, node.js, cordava 에 관해 배우고 간단한 프로그래밍을 통해 웹사이트를 설계하고 하이브리드앱을 구축한다.

◼ 이 과목은 웹 디자인을 전공하는 학생과 웹프로그래머가 되기를 원하는 학생들에게 필수추천 과목이며 그 외 웹사이트 구축하려는 학생들에게 좋은 교양과목이다.

데이터통계처리

◼ 본 강의를 통해서 통계학의 기본 개념을 학습하고, 사회 각 분야에서 응용되어 사용되고 있는 여러 가지 통계적 그래프나 통계적 결과를 바르게 이해하고, 바른 통계를 사용할 수 있는 능력을 익히도록 한다. 이를 위해 널리 사용되고 있는 통계 패키지인 SPSS를 통해 이루고자 한다.

오피스활용

◼ IT 시대의 기본 도구인 퍼스널 컴퓨터를 이해하고 사용하는 방법을 배운다. 퍼스널 컴퓨터를 활용하기 위해 운영체제인 OS(Windows-10) 그리고 컴퓨터 활용의 기본 툴인 MS-Office(Word, Excel, Powerpoint)를 익히고 실제 사용 가능하도록 연습한다. 퍼스널 컴퓨터를 활용하여 본인의 전공 공부와 생활에 필요한 기본 지식을 익힌다.
